gettimeofday

Synopsis:
          int gettimeofday(struct timeval *tv, void *tz);

Request:
‘Fgettimeofday,tvptr,tzptr’
Return value:
On success, 0 is returned, -1 otherwise.
Errors:
EINVAL
tz is a non-NULL pointer.
EFAULT
tvptr and/or tzptr is an invalid pointer value.
