14 Multi-threaded Applications

The following annotations report thread related changes of state.

^Z^Znew-thread
This annotation is issued once for each thread that is created apart from the main thread, which is not reported.


^Z^Zthread-changed
The selected thread has changed. This may occur at the request of the user with the thread command, or as a result of execution, e.g., another thread hits a breakpoint.
