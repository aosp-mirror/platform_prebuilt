3.10.5 Operators

The linker recognizes the standard C set of arithmetic operators, with the standard bindings and precedence levels: 

     precedence      associativity   Operators                Notes
     (highest)
     1               left            !  -  ~                  (1)
     2               left            *  /  %
     3               left            +  -
     4               left            >>  <<
     5               left            ==  !=  >  <  <=  >=
     6               left            &
     7               left            |
     8               left            &&
     9               left            ||
     10              right           ? :
     11              right           &=  +=  -=  *=  /=       (2)
     (lowest)

Notes: (1) Prefix operators (2) See Assignments.
